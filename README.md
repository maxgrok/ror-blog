# README

This is a Blog built with Ruby on Rails. 

* Ruby version = 2.5.3
* Rails version = 5.2.1

* Configuration 

add `.gitignore` files = config/master.key and config/credentials.yml.enc

* Database creation
  `rails db:create`

* Database initialization
  Make sure to be running Postgresql, otherwise app will not work. 

* Local Deployment

`rails s`
